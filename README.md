# Secure BI Delivery System

Monorepo for biometric-based delivery verification.

## Apps

- `backend`: Node.js + Express API
- `admin`: Admin dashboard scaffold
- `mobile`: React Native scaffold

## Quick start

1. Install dependencies from the project root:
   - `npm install`
2. Run backend in development:
   - `npm run dev:backend`

## API status

Implemented starter modules:
- Registration and password login
- Biometric enrollment and biometric login
- Delivery creation/assignment
- Proof-of-delivery capture and retrieval
